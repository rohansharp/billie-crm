## Integration Patterns

### 1. Slide-Over (Drawer) Pattern
Used for: Viewing detail without losing list context

- Collections Queue â†’ Account quick view
- Investigation â†’ Event detail JSON
- ECL Config â†’ Edit parameter modal

### 2. Modal Pattern
Used for: Write actions requiring focus

- Record Payment, Waive Fee, etc. (existing)
- Schedule Config Change
- Confirm Period Close

### 3. Wizard Pattern
Used for: Multi-step processes

- Period Close (5 steps)
- Export Creation (3 steps: type, options, confirm)

### 4. Tab Pattern
Used for: Related data in same context

- Account Detail (Overview, Transactions, Schedule, Fees, **Accruals**, **ECL**, Actions)
